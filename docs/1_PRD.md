# 1_PRD.md (JungleClear.gg 기능 정의서 v1.5)

## 1. 프로젝트 개요
* **서비스 목적:** 천상계 유저들의 '가장 빠른 초반 풀캠프 정글링 동선' 데이터와 유튜브 영상 제공.
* **핵심 가치:** 킬/외부 변수 없는 순수 PvE 벤치마크 데이터 선별.
* **타겟 유저:** 정글링 속도를 최적화하고 싶은 롤 유저.
* **데이터 선별 원칙:** 천상계(그랜드마스터/챌린저) 유저의 순수 PvE 풀캠프 클리어 기록만 제공한다. 킬/어시스트/데스 등 외부 변수가 개입된 경기는 제외되어, 순수한 정글링 실력만을 비교할 수 있다. *(기술적 선별 기준은 `9_DATA_STRATEGY.md` 참고)*

## 2. 유저 경험 정의 (UX)

> 이 섹션은 **유저가 보고, 느끼고, 행동하는 것**만 서술합니다. 구현 방법(컴포넌트명, CSS, 기술적 구조)은 `3_FRONTEND_IMPLEMENTATION.md`에 정의합니다.

### [화면 1] 메인 페이지 — "어떤 챔피언의 정글링 영상을 볼까?"

**첫 인상:**
* 유저가 사이트에 접속하면, 화면 중앙에 "MASTER THE JUNGLE"이라는 대형 타이틀과 검색창이 보인다.
* 검색창 아래에는 영상이 있는 인기 챔피언들이 카드 형태로 나열되어 있다.

**검색 경험:**
* 유저가 검색창에 챔피언 이름(한글)을 입력하면, 즉시 자동완성 결과가 드롭다운으로 나타난다.
* 영상이 있는 챔피언은 초상화 썸네일, 이름, 승률, Best 클리어 타임과 함께 "Watch Video" 버튼이 보인다. 클릭하면 해당 챔피언의 상세 페이지로 이동한다.
* 영상이 없는 챔피언은 흐릿하게(비활성) 표시되며 "영상 준비중" 문구가 보인다. 클릭할 수 없다.
* 영상이 있는 챔피언이 항상 목록 상단에, 없는 챔피언이 하단에 위치한다.

**챔피언 카드 탐색:**
* 검색창 아래에 인기순으로 정렬된 챔피언 카드들이 보인다. 영상이 있는 챔피언만 노출된다.
* 각 카드에는 챔피언 Splash 이미지, 한글 이름, 승률(%), Best 풀캠프 클리어 시간이 표시된다.
* 카드에 마우스를 올리면 재생 아이콘이 나타나며 카드가 살짝 떠오른다. 클릭하면 상세 페이지로 이동한다.
* 정렬 필터(All Tiers / Clear Speed / Win Rate)는 후속 구현 예정 *(→ `10_BACKLOG.md`)*

**공통 내비게이션:**
* 화면 최상단에 고정된 내비게이션 바가 있다. 좌측에 로고, 중앙에 Champions / Leaderboards 메뉴, 우측에 Login / Sign Up 버튼이 보인다.
* 모바일에서는 메뉴가 숨겨지고 로고와 인증 버튼만 보인다.

### [화면 2] 챔피언 상세 페이지 — "이 챔피언의 가장 빠른 정글링은?"

**첫 인상:**
* 유저가 챔피언을 선택하면, 좌측에 챔피언 정보, 우측에 영상이 보이는 2분할 화면이 나타난다.
* 가장 빠른 1위 기록의 영상이 자동으로 플레이어에 로드되어 있다.

**좌측 — 챔피언 정보 확인:**
* 챔피언의 Splash 아트 위에 이름과 이명(예: "눈먼 수도승")이 겹쳐서 표시된다.
* 그 아래에 WIN RATE, PICK RATE, BAN RATE 3가지 통계가 한눈에 보인다.
* 최단 클리어 시간과 해당 패치 버전, 정글 티어 순위(예: #4 Jungle)가 카드 형태로 표시된다.

**우측 — 영상 시청과 리더보드 탐색:**
* 화면 상단에 YouTube 영상 플레이어가 있다. 플레이어 아래에는 현재 영상의 기록 정보(시간, 패치 버전)가 보인다. 1위 영상에는 "RECORD HOLDER" 뱃지가 표시된다.
* 플레이어 아래에 리더보드가 있다. 각 항목에는 순위, 소환사명, 티어(Challenger/Grandmaster/Master), 클리어 시간, 패치 버전이 표시된다.
* 현재 재생 중인 항목은 시각적으로 강조되어 어떤 영상을 보고 있는지 바로 알 수 있다.
* 다른 순위의 항목을 클릭하면 해당 영상으로 플레이어가 즉시 전환된다.
* 영상이 1개뿐인 챔피언은 리더보드가 보이지 않는다.
* 조회수 표기, 패치/지역 필터는 후속 구현 예정 *(→ `10_BACKLOG.md`)*

**모바일 대응:**
* 모바일에서는 좌측(챔피언 정보)과 우측(영상)이 세로로 쌓여서 표시된다.
