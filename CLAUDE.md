# JungleClear.gg 프로젝트

## 세션 시작 시 필수 사항
작업을 시작하기 전에 반드시 아래 순서를 따를 것:
1. `docs/0_MASTER_GUIDE.md`를 읽고 프로젝트 맥락을 동기화
2. `docs/7_ROADMAP.md`를 읽고 현재 진행 상황 파악
3. **유저에게 현재 진행 상황과 다음 작업을 보고**한 후 작업 시작

이전 세션의 요약본이 있더라도 위 절차를 생략하지 않는다.

## 세션 종료 시 필수 사항
의미 있는 작업 단위가 완료되거나 세션이 종료될 때, `docs/7_ROADMAP.md`의 세부 진행 상황을 업데이트할 것.

## 절대 규칙
- **DDL(CREATE, ALTER, DROP TABLE 등)은 절대 실행하지 않는다.** DB 스키마 변경은 반드시 개발자가 직접 수행한다. SQL 파일(`backend/sql/`)의 작성/수정도 개발자 지시 없이 임의로 하지 않는다.
